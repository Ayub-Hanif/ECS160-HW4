## Getting Started
1. Clone this repo.
2. Run `sudo service redis-server start` to start Redis server.
3. Run `mvn clean install` in IntelliJ or terminal.
4. Run the app with:
   ```bash
   java -jar target/HW1-solution-1.0-SNAPSHOT.jar
   ```
